# 舞王僵尸文件清单

## 概述
游戏中有两种舞王僵尸类型：
1. **Disco 舞王** (Zombie_disco) - GOTY版本使用
2. **Jackson 舞王** (Zombie_Jackson) - 原版使用

两种舞王都使用相同的**伴舞僵尸** (Zombie_backup)，但伴舞僵尸有不同的头部贴图。

## 代码中的使用

### Reanimator.cpp 中的配置
```cpp
#ifdef _HAS_NEW_DANCERS
    { ReanimationType::REANIM_DANCER, "reanim\\Zombie_disco.reanim", 0 },
#else
    { ReanimationType::REANIM_DANCER, "reanim\\Zombie_Jackson.reanim", 0 },
#endif
```

### 关键区别
- **Disco舞王**: 使用 `Zombie_disco.reanim`，有更多装饰（眼镜、络腮胡、奖章等）
- **Jackson舞王**: 使用 `Zombie_Jackson.reanim`，更简洁的设计
- **伴舞僵尸**: 两种舞王共用 `Zombie_backup.reanim`，但头部贴图不同
